Pensiunea „Căsuța cu maci”

Description

„Căsuța Cu Maci” este construită special pentru a însufleți sufletul fiecărui călător care îi trece pragul. Tradițională și colorată, pensiunea este adaptată secolului XXI – unind astfel utilul cu plăcutul. Pensiunea din sat Poiana cu Cetate este alcătuită din: o cameră de zi cu loc de luat masa, o bucătărie complet utilată, o terasă generoasă și cinci dormitoare duble, cu baie proprie.

Location

Grajduri, sat Poiana cu Cetate
